# CurbNTurf Redesign: Site Vision

## Vision
To remodel the CurbNTurf platform to feel like "The Refined Explorer"—a premium but outdoorsy layout leveraging Three.js 3D models and an editorial, airy design system.

## Sitemap
- [x] index (`/`)
- [x] search / map (`/search`)

## Roadmap
1. Generate the **User Profile Dashboard**.
2. Generate the **Checkout Flow** for reserving an RV spot.
3. Replace the Home layout placeholder blocks with dynamic lists.

## Creative Freedom
- Interactive map components with unique map pins.
- Host revenue statistics graphs. 
- Custom image galleries for listings.
